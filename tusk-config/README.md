# Tusk Config

The **Tusk Config** package handles configuration loading and management for the Tusk Framework, supporting multiple formats and environments.

## Features
- **Multi-format Support**: Load configuration from PHP, JSON, YAML, or ENV files.
- **Environment Aware**: Seamlessly switch configurations based on the application environment.
- **Type-safe Access**: Retrieve configuration values with automatic casting and defaults.

## Installation
```bash
composer require tusk/config
```
