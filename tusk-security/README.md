# Tusk Security

The **Tusk Security** package provides essential tools for authentication, authorization, and data protection within the Tusk Framework.

## Features
- **Authentication**: JWT, Session, and API Key support.
- **Authorization**: Role-Based Access Control (RBAC) and Attribute-Based Access Control (ABAC).
- **Cryptography**: High-level wrappers for secure hashing and encryption.

## Installation
```bash
composer require tusk/security
```
