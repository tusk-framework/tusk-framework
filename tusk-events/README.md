# Tusk Events

The **Tusk Events** package implements the observer pattern and event dispatching logic for the Tusk Framework.

## Features
- **PSR-14 Compliant**: Follows the standard PHP event dispatcher specification.
- **Asynchronous Events**: Support for background event handling via the Tusk Engine.
- **Type-safe Listeners**: Automatic event-to-listener mapping.

## Installation
```bash
composer require tusk/events
```
