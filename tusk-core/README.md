# Tusk Core

The **Tusk Core** is the foundation of the Tusk Framework, housing the Dependency Injection container and the application lifecycle management.

## Features
- **IoC Container**: PSR-11 compliant dependency injection.
- **Provider System**: Modular bootstrapping of framework services.
- **Lifecycle Management**: Application startup and shutdown hooks.

## Installation
```bash
composer require tusk/core
```
